Getting started with WebAssembly and Rust

In order to compile Rust to WebAssembly we need do the following:

Requirements

  1. Install the standard Rust toolchain, including rustup, rustc and cargo
  2. wasm-pack
  3. cargo-generate
  4. npm

Instructions

  1. Clone this repo cargo generate --git https://github.com/rustwasm/wasm-pack-template and use the name "wasm-game-of-life" when asked.
  2. Change directory cd wasm-game-of-life and
  3. Run this command in the directory wasm-pack build.
  4. Run this command within the wasm-game-of-life directory: npm init wasm-app www.
  5. Open up wasm-game-of-life/www/package.json and edit the "dependencies" to include a "wasm-game-of-life": "file:../pkg" entry: 
    {
    // ...
    "dependencies": {
        "wasm-game-of-life": "file:../pkg", // Add this line!
    // ...
    }
}
  6. First, ensure that the local development server and its dependencies are installed by running npm install within the wasm-game-of-life/www subdirectory: npm install
  7. To take the package wasm-game-of-life and use it in a web page, you can use this Javascript template.
  8. Next, modify wasm-game-of-life/www/index.js to import wasm-game-of-life instead of the hello-wasm-pack package:
    import * as wasm from "wasm-game-of-life";

    wasm.greet();
  9. Since we declared a new dependency, we need to install it:

    npm install

    To serve locally: npm run start
  10. Within wasm-game-of-life/www run npm run start , you will see the Greeting message at localhost:8080.

Congratulations, this is the end of the tutorial!
